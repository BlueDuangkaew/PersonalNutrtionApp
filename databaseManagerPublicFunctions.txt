Food database manager:
    Search for food info by food name:
        Name:       find_food_name()
        Parameter:  Food name
        Output:     Dictionary

    Add new items to the existing database
        Name:       add_food()
        Parameter:  Dictionary

Eating history database manager:
    Search meal info by time :
        Name:       find_meal_date()
        Parameter:  Dictionary
        Output:     Dictionary
        
    Add new meal record:
        Name:       add_meal()
        Parameter:  Dictionary
